# 拆文系列（用于知乎/公众号/内网分享）

目标：把《企业如何落地 Agent-Native 架构》这篇长文拆成若干篇**可独立传播、可 3~5 分钟读完**的短文。

使用方式：

- 每篇建议 1500~3000 字（或 1 张核心图 + 1 个清单）。
- 每篇结尾固定加：
  - “完整版长文：README.md（仓库主页）”
  - “一页总览：README.md#one-page”
- 发布节奏：每周 2~3 篇，持续 3~4 周。
