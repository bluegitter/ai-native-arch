# 拆文系列：把长文拆成可分发的 10 篇短文

> 目标：每篇 1,500–3,000 字左右，能独立传播；每篇末尾统一回链到长文（仓库根目录 README）。

## 使用方式

- 你可以按顺序发布，也可以先挑最容易引流的 3 篇（推荐顺序见下）。
- 每篇末尾保留「延伸阅读：长文全文」链接，用短文为长文引流。

## 推荐发布顺序（更容易起量）

1. **01-卡点不在模型**（观点强、受众广）
2. **02-四个概念边界**（容易引发讨论）
3. **04-四原则**（可收藏、可转发）
4. **05-五层架构一页图**（架构师最爱）

其余篇章可作为系列持续输出。
